# CPManager

Proyecto CPManager — Backend (NestJS) + Frontend (React + Tailwind).

Estructura:
- backend/ — NestJS API
- frontend/ — React app

Cómo ejecutar localmente

Prerequisitos: Node.js 18+ y npm

1. Backend

cd backend
npm install
npm run start:dev

2. Frontend

cd frontend
npm install
npm start

GitHub
------
Para subir este proyecto a GitHub:

# Usando la CLI de GitHub (gh)
# 1. Autenticar: gh auth login
# 2. Crear repositorio remoto: gh repo create <owner>/<repo> --public --source=. --remote=origin
# 3. Push inicial: git push -u origin main

# Usando la web:
# 1. Crear un nuevo repositorio en github.com
# 2. Luego en tu máquina:
# git remote add origin <url>
# git branch -M main
# git push -u origin main

Notas: Ajusta la rama principal si usas 'master' en lugar de 'main'.
